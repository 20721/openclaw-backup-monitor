# 备份监控

系统备份、监控与自动恢复工具集合。

## 项目

### OpenClaw Backup Monitor

OpenClaw 网关备份监控系统 - 自动监控配置变化、异常自动恢复。

**功能：**
- 配置文件变更自动备份（保留最近3份）
- Gateway 异常自动检测与恢复
- Telegram 通知

**使用：**
```bash
cd openclaw-backup-monitor
bash install.sh
```

**文档：** [openclaw-backup-monitor](./openclaw-backup-monitor)
